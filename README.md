# 🎯 Number Guessing Game (Python)

## 📌 Case Study 1

### 📖 Objective
Create a Python program that allows the player to guess a randomly generated number within a given range. The program provides feedback on whether the guess is too high, too low, or correct.

---

## 🕹️ Game Description

The game generates a random number between **1 and 100**.  
The player attempts to guess the number.

After each guess, the program tells the player:

- 📈 "Too High" if the guess is higher than the target number  
- 📉 "Too Low" if the guess is lower than the target number  
- 🎉 "Correct!" if the guess matches the target number  

The game also counts the number of attempts taken to guess correctly.

---

## ⚙️ Features

✔ Random number generation  
✔ User input handling  
✔ High/Low feedback  
✔ Attempt counter  
✔ Option to play multiple rounds  
✔ Error handling for invalid inputs  

---

## 🧠 Concepts Used

- `while` loops  
- `if-elif-else` conditions  
- Functions  
- Exception handling (`try-except`)
